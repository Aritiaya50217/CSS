กำหนดตำแหน่งด้วย float

float กำหนดให้ Elemant สามารถลอยอยู่ด้านใดด้านหนึ่ง โดยค่าที่กำหนดได้ คือ

  • left,right ให้ลอยอยู่ทางด้านซ้ายและขวาตามลำดับ
  • inherit ลอยตาม parent element
  • none ไม่มีการลอย (default)

clear ไม่อนุญาตให้มีการลอยของ element โดยค่าที่กำหนดได้ คือ

  • left,right ไม่อนุญาตให้ลอยอยู่ทางด้านซ้ายและขวาตามลำดับ
  • both ไม่อนุญาตให้มีการลอยทั้งสองด้าน
