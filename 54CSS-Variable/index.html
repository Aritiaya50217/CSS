CSS Variable

  • การสร้างตัวแปรใน css เพื่อช่วยอำนวยความสะดวกในการกำหนด style ให้แต่ละ element โดยลดขั้นตอนการทำงานที่ซ้ำซ้อนใน css ให้โค้มีความเป็นระเบียบและง่ายต่อการจัดการมากยิ่งขึ้น
  • การกำหนดตัวแปรสามารถกำหนดชื่อได้เอง(custom variable) โดยขึ้นต้นด้วยเครื่องหมาย - ตามด้วยชื่อตัวแปร ซึ่งตัวแปรส่วนใหญ่ใน css จะนำมาเก็บค่าที่ใชเรียกทำงานซ้ำๆ เช่น สี เงา แอนิเมชั่น เป็นต้น
