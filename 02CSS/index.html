รูปแบบการประกาศใช้ CSS

แบบ Internal

  • Inline เป็นการแทรก attribute style ลงในแท็ก HTML เช่น

    • Test Inline

  • Embeded การกำหนดรูปแบบ style sheet ภายในไฟล์เดียวกันกับไฟล์เว็บ เช่น

    • <style>
      p {
        color:red;
        text-align:center;
      }
    </style>

แบบ External

  • กำหนดไฟล์ style sheet แยกเป็นอีกไฟล์โดยมีนามสกุล .css แล้วค่อยเชื่อมเข้ามาภายในเว็บเพจ เหมาะสำหรับการกำหนดให้หลายๆเว็บเพจใช้รูปแบบ style เดียวกันผ่านการเขียน style แค่ครั้งเดียว เช่น

    • style.css

    <style>
      p {
        color:red;
        text-align:center;
      }
    </style>

    นำมาใช้ในไฟล์ html

    <head>
    <title> </title>
    <link rel="stylesheet" href="style.css">
    </head>
