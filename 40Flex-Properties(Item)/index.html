Flex-properties (Items)

  • flex-shrink : ให้ item หดตัวจำนวนเท่าใดเมื่อเทียบกับ item อื่น ๆ ค่าเริ่มต้นเป็น 1
  • flex-grow : ให้ item ขยายจำนวนเท่าใดเมื่อเทียบกับ item อื่น ๆ ค่าเริ่มต้นเป็น 0
  • flex-basis : กำหนดค่าความยืดหยุ่นเริ่มต้น
  • flex : 1 ทำให้ item ที่อยู่ในแถวเดียวกันมีขนาดเท่ากัน
