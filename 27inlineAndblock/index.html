Display Inline,Block, Inline-Block

display คือ การจัดรูปแบบการแสดงผลข้อมูล layout

  • none ไม่มีการแสดงผล
  • block แสดงผลแบบ block โดยการขึ้นบรรทัดใหม่ก่อน (เรียงในแนวตั้ง)
  • Inline แสดงผลแบบ inline โดยไม่มีการขึ้นบรรทัดใหม่ (เรียงในแนวแนว)
  • Inline-Block แสดงผลแบบแนวนอนและขยายพื้นที่ด้านใน
