การจัดตำแหน่งด้วย Position

position ใช้ในการจัดตำแหน่งของวัตถุ

  • static จัดวางแบบปกติเป็นค่า default
  • relative ใช้ในการจัดวางและกำหนดตำแหน่งวัตถุโดย การนับจากจุดที่วัตถุนั้น ๆ อยู่
  • absolute ใช้ในการจัดวางวัตถุ ให้ไปอยู่ตำแหน่งใดก็ได้แต่ต้องกำหนดจากขอบของ element ที่บรรจุ วัตถุนั้น ๆ อีกที (ถ้าไม่มีอะไรครอบก็ถือว่า body เป็น element ที่ครอบวัตถุที่ระบุ absolute)
  • fixed ใช้ในการจัดวางวัตถุให้อยู่ตำแหน่งเดิม
  • sticky ให้วัตถุติดขอบเมื่อเลื่อนไปถึง
